# Ticket Management System

This project is a Python and MySQL-based ticket management program created for a CIS course assignment.

## Features
- Check if a ticket number exists
- Check ticket validity based on date
- Calculate total ticket prices
- Restart or quit the program from a menu

## Technologies Used
- Python
- MySQL
- AWS RDS
- GitHub

## Purpose
This project demonstrates basic database connection, SQL queries, user input handling, and menu-based program logic.
